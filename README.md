PSR-15 Middleware collection implemented by react promises

## Install

Via Composer

```bash
$ composer require antidot-fw/antidot-react-psr15:dev-master
```

## Simple Usage

```php
<?php
// router/middleware.php
declare(strict_types=1);

```
